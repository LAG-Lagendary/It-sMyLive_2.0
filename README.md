It-sMyLive_2.0

Этот репозиторий предназначен для автоматической настройки рабочей среды Linux после чистой установки. Использует Ansible для быстрой установки необходимого софта и конфигурации системы.
Как пользоваться

После того как вы установили свежий Linux, выполните следующие шаги в терминале:

1. Установка Git

Если Git еще не установлен в системе:

Bash

sudo apt update && sudo apt install git -y

2. Клонирование и запуск

Клонируйте репозиторий и запустите скрипт автоматической настройки:
Bash

git clone https://github.com/LAG-Lagendary/It-sMyLive_2.0.git
cd It-sMyLive_2.0
chmod +x setup.sh
./setup.sh
